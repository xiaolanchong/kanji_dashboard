栲[tae] арх.
1. ткань из древесных волокон
(бумажного дерева,
глицинии, конопли и т. п. )
2.  общее название для любой ткани
栲[taku] арх. бумажная шелковица См. 楮
В сочетаниях:

1) ткань, материя («tae»)

白栲[shirotae] арх. белая ткань
荒栲[aratae] арх. грубая ткань (из конопли и т. п. )
和栲[nigitae] арх. мягкая ткань из тонких волокон

2) бумажная шелковица («taku»)

栲縄[takunawa]верёвка из волокон бумажной шелковицы
栲布[takununo]ткань из волокон бумажной шелковицы
